git clone  <url> ====create the repository to our local workspace

git status== to check whether git has initialized in ur current directory

git add <filename> === add the file "filename1" to staging area (to perform commit and push)

git add .===add all the files in the directory to the staging area

git commit -m "message"=== to acommit that it has been tracked by git

git push -u origin main====to push the commited file to the workspace

git init==to initializ git in a folder
